# Touchless Button with vl6180x ToF using Arduino

## library

[pololu-vl6180x 공식 라이브러리](https://github.com/pololu/vl6180x-arduino)

## Pin info

5V Arduino boards

| Arduino | VL6180X board |
| ------- | ------------- |
| 5V      | VIN           |
| GND     | GND           |
| A4      | SDA           |
| A5      | SCL           |
